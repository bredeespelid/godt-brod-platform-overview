System architecture

One Replit-hosted server.
Three Fabric endpoints. PostgreSQL in the middle.

A single Express server on Replit serves the React frontend, the REST API, and the Azure AD session layer. PostgreSQL holds operational state. Microsoft Fabric SQL Analytics Endpoints provide the enterprise data backbone for finance and operations.

Technology stack

End-to-end TypeScript with shared schemas.

Drizzle ORM and Zod schemas live in a shared package, so the type contract between frontend and backend never drifts. One language, one type system, one source of truth.

Operational design

Production patterns, not prototype shortcuts.

Tiered caching, scheduled jobs, retries with backfill, structured failure handling, row-level security on certified Power BI datasets, and CI/CD via GitHub Actions. Built like a system that has to keep working at 06:00 on a Monday.
